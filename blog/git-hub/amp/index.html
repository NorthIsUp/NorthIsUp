Made this alias to open github urls in my browser. I’m using the python webbrowser module to launch the default browser. [alias] hub = !git remote -v|sed -e 's@.*git:\\(.*\\).git.*@http:\\1@'|xargs python -c "'import sys,webbrowser as w;w.open(sys.argv[1]);'" Turns out I missed some cases with my previous sedexpression, this one is better. ` [alias] #os x (faster): hub = !git remote -v sed “‘s .git[:@]\(.\).git.* http://\1 ;s m: m/ ’” uniq xargs open #platform independent (slower, but it works on linux too): hub = !git remote -v sed “‘s .git[:@]\(.\).git.* http://\1 ;s m: m/ ’” uniq xargs python -c “‘import sys,webbrowser as w;w.open(sys.argv[1]);’” ` If you want to launch slightly faster you can avoid launching a python process by calling into your browser directly.
